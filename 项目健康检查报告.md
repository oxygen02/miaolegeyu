### 4.1 常见问题扫描

#### 🔴 内存泄漏：`vote.js` 的 `onUnload` 为空

```
文件：pages/vote/vote.js
行号：346-347
问题：onUnload() 函数体为空，页面卸载时未清理 setTimeout 定时器
影响：用户从投票页返回后，定时回调可能尝试对已卸载的页面 setData，导致错误或内存泄漏
修复建议：
```

```javascript
// pages/vote/vote.js
onUnload() {
  // 清理所有 setTimeout 定时器
  if (this._timers) {
    this._timers.forEach(t => clearTimeout(t));
    this._timers = [];
  }
},
```
