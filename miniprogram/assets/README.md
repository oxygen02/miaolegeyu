# 图标资源说明

## 需要的图标文件

以下图标需要添加到 `/assets/` 目录：

### 1. icon-close.png (28x28px)
关闭按钮图标，灰色 X 符号

### 2. icon-download.png (32x32px)
下载/保存图标，用于海报保存按钮

### 3. icon-share.png (32x32px)
分享图标，用于海报分享按钮
